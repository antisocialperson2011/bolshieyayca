# 🎉 Bolshieyayca Giveaway Bot

Telegram-бот, созданный для организации розыгрыша (конкурса) среди аудитории пользователя, производящий случайный и честный выбор среди списка участвующих. Бот реализован для упрощенного создания розыгрышей в различных Telegram-каналах, не требуя от пользователя много времени. Функции и логическая работа бота созданы на основе двух блок-схем.

## Структура проекта

```
giveaway_bot/
├── bot.py                  # Основной скрипт, запускающий бота
├── config.py               # Конфигурация (токен, список администраторов)
├── requirements.txt
├── database/
│   └── db.py               # База данных на основе SQLite
├── handlers/
│   ├── start.py            # Реализация начала работы пользователя с ботом
│   ├── giveaway_entry.py   # Реализация начала работы пользователя с розыгрыша с ботом 
│   ├── giveaway.py         # Создание и управление розыгрышами
│   ├── channels.py         # Управление каналами
│   └── admin.py            # Админ-панель
└── utils/
    └── keyboards.py        # Реализация управления ботом при помощи кнопок Inline Keyboard
```

## Возможности бота
⚪ **Работа с пользователем**

 - Автоматическая регистрация пользователя в боте при первом запуске
 - Хранение внесенных пользователем данных
 - Возможность использовать меню бота при помощи InlineKeyboard
 - Добавление пользователя в список участников определенного розыгрыша, в случае перехода по кнопке с розыгрыша
 
 ⚪ **Создание и последующее управление розыгрышами**
 
 - Создание розыгрыша с настройками параметров: описание розыгрыша; название кнопки для участия; канал, где будет опубликован розыгрыш; условия окончания розыгрыша - по времени или по количеству участников (в случае первого варианта - также выбор времени окончания розыгрыша)
 - Досрочное завершение ранее созданного розыгрыша в личном кабинете бота
 
 ⚪ **Админ-панель для управления и слежки действий внутри бота**
 
 - Возможность блокировки или разблокировки Telegram-канала при различных обстоятельствах (нарушение каких-либо правил, образец для теста поведения бота, различные ошибки)
 - Возможность блокировки или разблокировки пользователя бота при различных обстоятельствах (подобные были указаны выше)
 - Возможность проверки логов для различных ситуаций (просмотр новых пользователей, добавленных или удаленных каналов, созданных или отмененных розыгрышей и прочее)

## Функции и логика бота

**Начало работы бота (в первый раз или в другие последующие)**
 - Пользователь использует бота, отправив команду /start
 - Проверка регистрации
	 - Пользователь использовал бота ранее -> переход в главное меню
	 - Пользователь впервые использует бота -> добавление пользователя в БД -> переход в главное меню
 - Показ главного меню
	 - Создать розыгрыш
	 - Мои розыгрыши
	 - Мои каналы
 
 **Создание розыгрыша**

 - Пользователь нажал кнопку "Создать розыгрыш" в главном меню
 - Проверка добавленных каналов
	 - 1 или более Telegram-каналов было добавлено пользователем -> переход функции далее
	 - 0 Telegram-каналов было добавлено пользователем -> вывод сообщения об отсутствующих добавленных каналов -> конец функции
 - Ввод параметров для опубликования розыгрыша
	- Описание розыгрыша (будет указано в виде сообщения в посте)
	- Выбор названия кнопки, по которой будут переходить участвующие
	- Выбор канала, в котором будет проводится розыгрыш
	- Выбор условия окончания розыгрыша
		- Пользователь выбрал окончание розыгрыша по количеству участников -> переход функции далее
		- Пользователь выбрал окончание розыгрыша в определенное время -> выбор пользователем время окончания розыгрыша -> переход функции далее

	 - Уведомление об успешном опубликовании розыгрыша

		 -	Вывод розыгрыша в канале с установленными пользователем ранее параметрами
		 -	Вывод сообщения пользователю об успешном опубликованном розыгрыше в канале
		 
**Просмотр уже созданных ранее розыгрышей**

 - Пользователь нажал "Мои розыгрыши" в главном меню
	 - Пользователь не имеет ранее созданные активные розыгрыши -> вывести сообщение об отсутствии созданных или активных розыгрышей -> конец функции 
	 - Пользователь имеет ранее созданные активные розыгрыши -> вывести список розыгрышей и кнопку возврата
		 - Пользователь выбрал кнопку возврата -> вернуться в главное меню
		 - Пользователь выбрал один из активных розыгрышей -> открыть розыгрыш и вывести: ID Telegram-канала, в котором опубликован розыгрыш, условие окончания розыгрыша, кнопку "Отменить розыгрыш", кнопку возврата в список розыгрышей
			 - Пользователь нажал кнопку возврата -> вернуться к списку активных розыгрышей
			 - Пользователь нажал на кнопку отмены розыгрыша -> удалить пост с розыгрышем в канале пользователя, вывести сообщение об успешной досрочной отмене розыгрыша -> конец функции
			
**Добавление новых каналов или просмотр добавленных ранее**

 - Пользователь нажал "Мои каналы" в главном меню
	 - Пользователь уже имеет добавленные каналы -> вывести список каналов, кнопку добавления нового канала, кнопку возврата
		 - Пользователь нажал кнопку возврата -> вернуться в главное меню
		 - Пользователь нажал на один из каналов -> вывести информацию о добавленном канале, кнопку удаления канала, кнопку возврата
			 - Пользователь нажал кнопку возврата -> вернуться в список каналов
			 - Пользователь нажал кнопку удаления канала -> удалить канал из списка каналов пользователя, вывести сообщение об успешном удалении канала
		- Пользователь нажал кнопку добавления канала -> вывод сообщения о добавлении бота на канал с правами администратора и последующего ввода сообщением ссылки на канал в личные сообщения бота, вывод кнопки возврата
			- Пользователь нажал кнопку возврата -> вернуться в список каналов
			- Пользователь указал ссылку на канал -> проверка на выполненные условия (если условия выполнены: вывод сообщения об успешном добавлении канала -> конец функции; если условия не выполнены или выполнены частично: вывод сообщения о невыполненных условиях -> ожидание повторого ввода, либо конец функции)
	- Пользователь не имеет добавленных каналов -> вывод сообщения об отсутствии каналов, вывод кнопки добавления нового канала, вывод кнопки возврата
		- Пользователь выбрал кнопку возврата -> вернуться в главное меню
		- Пользователь нажал кнопку добавления канала -> вывод сообщения о добавлении бота на канал с правами администратора и последующего ввода сообщением ссылки на канал в личные сообщения бота, вывод кнопки возврата
			- Пользователь нажал кнопку возврата -> вернуться в список каналов
			- Пользователь указал ссылку на канал -> проверка на выполненные условия (если условия выполнены: вывод сообщения об успешном добавлении канала -> конец функции; если условия не выполнены или выполнены частично: вывод сообщения о невыполненных условиях -> ожидание повторого ввода, либо конец функции)


## Установка

1. Прописать в командной строке или в терминале следующую строку:
```
pip install -r requirements.txt
```

## Настройка

1. Откройте `config.py`
2. Вставьте токен бота в `BOT_TOKEN`
3. Добавьте Telegram user_id администраторов в список `ADMIN_IDS`

```python
BOT_TOKEN = "123456789:AABBcc..."

ADMIN_IDS = [
    123456789,   # ваш user_id
]
```

Чтобы узнать ID пользователя, требуется использовать @userinfobot в Telegram.

## Запуск

1. Открыть скрипт `bot.py` или воспользоваться командами ниже
```bash
python bot.py
# или
BOT_TOKEN=ваш_токен python bot.py
```

## База данных

SQLite файл `giveaway_bot.db` создаётся автоматически при первом запуске.

Таблицы:
- `users` — пользователи
- `channels` — каналы
- `giveaways` — розыгрыши
- `participants` — участники розыгрышей
- `logs` — лог событий
